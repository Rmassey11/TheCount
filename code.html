The Code for The Count


                    //Get the values from the page
//starts or controller function
function getValues(){
    //get values from the page
    let startValue = document.getElementById("startValue").value;
    let endValue = document.getElementById("endValue").value;

    //We need to validate our input
    //parse into Integers
    startValue = parseInt(startValue);
    endValue = parseInt(endValue);

    if(Number.isInteger(startValue) && Number.isInteger(endValue)){
        //call generateNumbers
        let numbers = generateNumbers(startValue, endValue);
        //call displayNumbers
        displayNumbers(numbers);

    } else {
        alert("You must enter integers");
    }

}

//Generate number from startValue to endValue
//logic function(s)
function generateNumbers(sValue, eValue){


    let numbers = [];

    //we want to get all numbers from start to end

    for(let index = sValue; index <= eValue; index++){

        //this will execute in a loop until index = eValue
        numbers.push(index);
    }


    return numbers;

}

//display the numbers and mark even numbers bold
//display or view function
function displayNumbers(numbers) {

    let templateRows = "";
    for (let index = 0; index < numbers.length; index++) {

        let className = "even";
        let number = numbers[index];

        if(number % 2 == 0){
            className = "even";
        }
        else{
            className = "odd"
        }
        //This does not render correctly with Prism, see the source
        templateRows += `${number}`;
    }

    document.getElementById("results").innerHTML = templateRows

}

The Code is structured in three functions.

getValues

The getValues function pulls the values from the page for use in our other functions. We declare the variables startValue and endValue which are Block Scoped into this code block. These values are then changed from Strings into Integers. A validation is then performed to ensure that the values can become Integers, if not an error Alert is thrown and the function is terminated. If the validation passes we declare the variable numbers(also Block Scoped) which in turn calls the function generateNumbers with the startValue and endValue variables passed into it. Once that function returns we then call the displayNumbers function with the numbers variable passed into it that is returned from generateNumbers.

generateNumbers

The generateNumbers function takes the values gathered by getValues and creates an array of all the Integers from the startValue to the endValue. We changed the values passed in from getValues to sValue and eValue to show that values can be named anything the creator wishes from one function to the next. To begin this function we declare the Block Scoped variable numbers as an empty array. We then create a "for loop" that begins at the sValue, increases by 1 each time, and ends at eValue. Once the loop exits the function returns the array as the variable numbers back into the getValues function to then immediately be entered as the parameter for the displayNumbers function.

displayNumbers

The displayNumbers function takes the array provided by generateNumbers and creates a row in the table provided in our HTML with each Integer it contains. First we declare a Block Scoped variable templateRows as an empty String. Next we run another "for loop" that iterates the entire length of our array. During each iteration of our loop we declare a Block Scoped variable className and set it to the string "even" and the variable number which is set to the current Integer of the array that our loop is at. Next we check to see if the current Integer is even or not. If it is even the className variable remains as the string "even". If the current Integer is odd we change the className variable to "odd". After the className variable is decided we then use a template literal that tells the HTML to create a new table row in its table. This row is assigned a class of "even" or "odd" depending on the check just performed. It is then given its table data as the current Integer of our array. If the class "even" is given then our css file makes the number appear bold and large in our table.
